This

La palabra reservada `this`, hace referencia al ámbito dentro del cuál la función fue llamada y cambia dependiendo de cómo se llame la función. Nunca se refiere a la función misma.

Hay varias reglas sencillas para definir a qué está refiriendo la palabra `this`:

  • Si fue llamada como una función independinte, se refiere al entorno global o a la ventana. En modo estricto esta forma está prohibida, genera un error.
  • Si fue llamada dentro de una instancia de objeto, se refiere a la instancia de objeto.
